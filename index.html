display:flex

A propriedade display: flex deve ser a primeira a ser declarada no CSS. A partir dela é que o CSS vai entender que precisa estilziar o contéudo do contêiner com a caixa flexível (Flexbox)

Valores padrão das propriedades do Flexbox
propriedade valor
flex-direction row
justify-content flex-start
align-content flex-start

justify-content

A propriedade justify-content pode ser usada para justificar o conteúdo do elemento contêiner em seu eixo primário, ou seja, eixo x (row, row-reverse) ou eixo y (column, column-reverse).

center

O valor "center" vai alinhar o centeúdo ao centro do eixo primário.

Exemplo:

Center

flex-start

A propriedade flex start alinha os contêiners para o lado esquerdo, o ínicio dela.

Exemplo:

Flex Start

flex-end

A propriedade flex start alinha os contêiners para o lado direto, o final dela.

Exemplo:

Flex end

space-beetwen

A propriedade space-beetwen faz com que os itens tenham um espaçamento proporcional de um ao outro.

Exemplo:

Space Beetwen

space-around

A propriedade space around faz com que o item inicial e final tenham o espaçamento igual, enquanto os demais é divididos iguais, mas com espaçamento divergente do inicial e final.

Exemplo:

Space Around

space-evenly

A propriedade space evenly faz com que o espaçamento seja dividido de forma igual para todos os itens.

Exemplo:

Space Evenly

Initial

A propriedade Initial faz com que o item volte para o padrão.

Exemplo:

Initial

inherit

A propriedade Inherit herda a mesma propriedade do elemento pai.

Exemplo:

Inherit

align-items

O align-items nos permite fazer alterações no alinhamento dos itens do flexbox, podendo mudar para cima, para baixo, mas sempre é mantido no eixo X (na horizontal).

O align-items-stretch nos permite fazer alterações no alinhamento dos itens do flexbox, podendo mudar para cima, para baixo, mas sempre é mantido no eixo X (na horizontal).
